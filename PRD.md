# LegalFriend - AI-Powered Lease Contract Review Assistant

## Product Overview
LegalFriend is a web application designed to help users understand and review lease contracts using AI technology. The platform provides automated contract analysis, highlighting key terms, potential issues, and offering explanations in plain language.

## Core Features

### 1. Contract Review
- Upload lease contracts in various formats (PDF, DOC, DOCX)
- AI-powered analysis of contract terms and conditions
- Identification of key clauses and potential red flags
- Plain language explanations of legal terms

### 2. User Dashboard
- View contract review history
- Track review status
- Access saved analyses
- Download reviewed contracts with annotations

### 3. Educational Resources
- Library of common lease terms
- Guides on tenant rights
- FAQ section
- Blog posts with legal tips

### 4. Interactive Features
- Highlight and comment on specific contract sections
- Ask questions about specific clauses
- Save and export annotations
- Share contract reviews with others (optional)

## Technical Architecture

### Frontend
- React with TypeScript
- Tailwind CSS for styling
- Shadcn UI components
- Responsive design for mobile and desktop

### Backend (Supabase)
- User authentication and authorization
- Contract storage and management
- Analysis results persistence
- Blog content management

### AI Integration
- Contract parsing and analysis
- Term extraction and explanation
- Risk assessment
- Question answering

## User Flow
1. User signs up/logs in
2. Uploads contract for review
3. Receives automated analysis
4. Can interact with results and ask questions
5. Saves or exports the review
